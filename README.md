# Складской учёт — GitHub Pages версия

Это простое веб-приложение, которое можно залить в GitHub и открыть на iPhone как приложение.

## Что умеет
- добавлять позиции склада
- вести остатки
- считать статусы: НОРМА / МАЛО / КУПИТЬ / НЕТ
- собирать список закупки
- хранить данные локально в браузере
- экспортировать и импортировать JSON

## Как запустить через GitHub Pages
1. Создай новый репозиторий на GitHub.
2. Загрузи в корень репозитория файлы:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. В GitHub открой `Settings` → `Pages`.
4. В `Build and deployment` выбери публикацию из ветки `main` и папки `/root`.
5. Сохрани настройки.
6. Через минуту у тебя будет ссылка вида:
   `https://username.github.io/repository-name/`

## Как поставить на iPhone
1. Открой ссылку в Safari.
2. Нажми `Поделиться`.
3. Выбери `Добавить на экран Домой`.
4. Включи `Открывать как веб-приложение`, если Safari это покажет.

## Важно
Данные хранятся в браузере на конкретном устройстве. Если хочешь резервную копию — экспортируй JSON.
